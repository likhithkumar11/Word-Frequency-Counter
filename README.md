# Word Frequency Counter

## Description
This Python program reads a text file, cleans the text, converts everything to lowercase, removes punctuation, splits into words, counts word frequencies, and displays useful statistics. It also handles invalid file paths, optional filtering, and printing results in sorted order.

## Features
- Prompts user for input file path
- Counts total words and unique words
- Shows top N most frequent words
- Removes punctuation and special characters
- Converts all words to lowercase
- Optional: ignore short words, stop-word removal, alphabetical listing
- Handles invalid inputs gracefully

## Requirements
- Python 3.x
- No external libraries required
